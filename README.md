# Movie Exploratory Data Analysis (EDA)

## Project Overview

This project explores a movie dataset using Python and Pandas.

The goal is to clean the dataset, perform exploratory data analysis (EDA), and discover insights about movie genres, ratings, popularity, languages, and release years.

---

## Dataset Features

The dataset includes the following information:

- Title
- Release Date
- Genre
- Original Language
- Popularity
- Vote Count
- Vote Average
- Overview
- Poster URL

---

## Objectives

This project answers the following questions:

- What are the most common movie genres?
- Which genres have the highest average ratings?
- Which genres receive the highest average vote counts?
- How are movie ratings distributed?
- How is movie popularity distributed?
- Is there a relationship between popularity and movie ratings?
- How has movie production changed over the years?

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights

---

## Key Findings

- Drama is the most common genre.
- Most movie ratings are between 6 and 7.
- Popularity is highly right-skewed.
- There is almost no linear relationship between popularity and average rating.
- History and War have the highest average ratings.
- The number of released movies generally increased over time.
